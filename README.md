holmes-api
==========

API for holmes website validation.

Waffle
------

https://waffle.io/holmes-app/holmes-api

Requirements
------------

* MySQL
* Python
* python-pip
* Redis

How to install
--------------

The holmes-api software has a Makefile to help with common tasks. To install, just type:

    make setup

To install data/migrations:

    make data
